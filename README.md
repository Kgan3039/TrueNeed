# TrueNeed

TrueNeed is a mobile platform that connects people offering items with people requesting them through intelligent real-time matching.

## Built With

- React Native
- Expo
- Firebase
- Firestore
- Firebase Authentication
- TypeScript

## Features

- User authentication
- Offer and request system
- Real-time match generation
- Match inbox
- Impact dashboard
